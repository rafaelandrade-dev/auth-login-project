# 🛠️ Backend Simulado – JSON Server

Este repositório contém um **backend fake** baseado em [JSON Server](https://github.com/typicode/json-server).  
Ele será usado como **API simulada** para a avaliação de frontend.  

O objetivo é fornecer **rotas REST** (CRUD de usuários) e **login com JWT falso**, sem necessidade de banco de dados ou backend real.

---



## 📦 Requisitos

- [Node.js](https://nodejs.org/) (versão 16+ recomendada)
- npm (vem junto com o Node)

---

## 🚀 Como rodar

1. Instale as dependências:

```bash
npm install
```

2. Rode o projeto: 

``` bash
npm start
